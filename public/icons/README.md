# PWA Icons

Bu klasöre aşağıdaki dosyaları yerleştir:

- `icon-192.png` — 192x192 (PWA standart)
- `icon-512.png` — 512x512 (PWA standart)
- `icon-maskable.png` — 512x512, maskable safe zone ile (Android adaptive)

**Not:** QRMasa platform ikonu buraya gelir (genel fallback).
Her işletmenin kendi ikonu Firestore `venues/{id}/branding` altından runtime'da yüklenir.

**Hızlı üretim için:** https://maskable.app veya https://realfavicongenerator.net
